# ArchiTech Name and Branding Policy

This policy applies to the names, logos, icons, visual identity, domain names,
and other source-identifying elements associated with the ArchiTech project
(collectively, the **ArchiTech Marks**).

The source code in this repository is licensed under the GNU General Public
License version 3 only (`GPL-3.0-only`). That license grants rights to the
covered source code. It does not grant permission to use the ArchiTech Marks
in a way that suggests sponsorship, endorsement, affiliation, or official
status.

## Permitted uses

You may:

- refer to the official project by its name in factual statements;
- state that a project is based on or derived from ArchiTech Launcher;
- link to the official ArchiTech website or repository;
- reproduce the ArchiTech Marks in reviews, news, commentary, screenshots,
  or other nominative and informational uses where permitted by law.

Any such use must be accurate and must not imply that a modified project is an
official ArchiTech release.

## Forks and derivative projects

A fork or derivative distribution that is not published by the ArchiTech
project must:

- use a clearly different product name;
- replace the ArchiTech logo, application icon, and other official branding;
- avoid describing itself as the official ArchiTech Launcher;
- avoid using ArchiTech-controlled domains, social accounts, package listings,
  or release channels;
- clearly identify that it is an independent modified version.

A factual statement such as “Based on ArchiTech Launcher” is permitted when it
is not presented more prominently than the derivative project's own name and
does not create confusion about origin or endorsement.

## Prohibited uses

Without prior written permission, you may not:

- use the ArchiTech Marks as the primary name or branding of a modified build;
- register confusingly similar domains, social accounts, application listings,
  or package names;
- use the ArchiTech Marks to imply approval, partnership, sponsorship, or
  affiliation;
- use the ArchiTech Marks in a deceptive, fraudulent, or misleading manner.

## No restriction on GPL rights

This policy applies only to names, branding, and source-identifying marks. It
does not limit the rights granted to the source code under `GPL-3.0-only`.

Permission requests should be directed to the repository owner through the
official ArchiTech project channels.
